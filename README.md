心臓マッサージ後、およそ13～14秒後に心細動イベントが発生するので、それまでの間に上記工程を全て行います。ピンセットで掴めなかったりすると間に合わないぐらい余裕がありません。

運が良ければ心細動が発生しないこともあります。発生した場合はおよそ10秒程のロスです。

### 2-3 Dependable Pair/頼りになる者 (ヴァレリー)

2回目のカウンターショック後、骨が出てくるまでの演出の間に超執刀が使えるので使用します。

### 2-4 Stigma/スティグマ

スティグマ：ケイル

ケイルにレーザーを当てると逸れるように若干後退するので、レーザーは押しっぱなしにせず、ほどよく連打するように撃ちます。

最初のケイルを処理すると2体動時に現れます。2体の位置が近いときはレーザーで薙ぎ払うようにすると、レーザー消費は増えますが処理速度を上げられます。

### 2-5 Lost in the Flames/秘密は炎に燃え

皮膚培養用のマスは12個、火傷一つにつき4枚必要なので全マス利用で火傷3つまで処理できます。そして火傷総数は9個なので3セットで済ませられます。

火傷は一定時間毎に出血するようになっていて、皮膚を乗せていても出血した場合は全て剥がれてしまいます。通常は出血したのを確認してから皮膚を乗せますがSpeedrunでは待っている時間が惜しいので自分の操作速度に合わせてどの火傷から処理するか決めましょう。

最初から黒くなっている火傷に関しては出血しなくなるので最後まで放置します。

### 3-3 Collaborative Research/共同研究

心筋パッチのガイドは収縮を2回繰り返した後に角度を変えて2回収縮するのを繰り返します。

## 3-4 Cursed Burden/忌まわしき痕

赤い腫瘍に関してはコアへレーザー攻撃した後から再出現するまでの間に処理できるので放置しても大丈夫ですが、バイタルはそれなりに削られます。バイタルと相談して放置するかどうか決めましょう。

## 3-5 The Deputy Secretary/DHS副総監 （ヴァレリー）

最初の動脈瘤に関しては道具を注射器に合わせていると鎮静剤の瓶が現れないので注射器以外の道具を選択しておきましょう。

また、動脈瘤は出現直後であれば鎮静剤満タンで2個まで切除まで持っていけます。注射ミスで中身を減らさないように注意しましょう。

最初の動脈瘤を処理すると次の動脈瘤出現まで時間があるのでやや左上に移動して注射器に消炎剤を充填しましょう。

2個の動脈瘤が出現します。残り1個を縫合する段階まで処理を進めるとさらに2個が出現します。

6個の動脈瘤を処理すると時間を置いて全体に4個の動脈瘤が出現するので事前にヴァレリーで超執刀を使い、さらに鎮静剤を充填しておきます。

左下の動脈瘤を残して他の動脈瘤を処理します。



全て処理すると動脈瘤が4個同時に出現します。左下が最も早く破裂するので超執刀を使いつつ左下以外を処置して破裂を狙いましょう。
